Sketch

Home | Classes | Methods | Properties

** Location:**Sketching & Drawing Sketch

Namespace: AlibreScript.API | Kind: Class

The Sketch class provides functionality for 2D sketching operations including creating geometric figures, applying constraints, and managing sketch elements.

Properties

Figures

Type: Object A list of figures (line, circle, circulararc, bspline, ellipse, elliptical arc) defined on the sketch

Name

Type: Object Name of the sketch

Origin

Type: Object The point that defines the origin
